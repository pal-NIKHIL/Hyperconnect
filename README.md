
# Hyperconnect

Our social media website is built using the MERN (MongoDB, Express.js, React.js, Node.js) stack, along with additional frameworks and libraries.
## Backend:

- Express.js
- MongoDB
- Mongoose
- bcrypt
- body-parser
- cors
- dotenv
- helmet
- jsonwebtoken
- morgan
- multer
- gridfs-stream

## Frontend:

- React.js
- React Router
- Redux
- @reduxjs/toolkit
- @mui/material
- @mui/icons-material
- formik
- yup
- react-dropzone
- @emotion/react
- @emotion/styled
